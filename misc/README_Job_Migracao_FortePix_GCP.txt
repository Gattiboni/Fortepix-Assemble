FortePix – Job de Migração para Google Cloud com Antifraude, 2FAT e Biometria
=============================================================================

Este pacote foi preparado para apoiar a migração da nova versão da API FortePix
para o Google Cloud (Console), utilizando Cloud Run + Artifact Registry, incluindo
códigos de exemplo para:

- Módulo Antifraude (análise de risco + cruzamento de dados)
- Módulo 2FAT (dois fatores com código temporário)
- Módulo 2FAT + Biometria (validação combinada)
- Cruzamento de dados atualizado (chaves, histórico, perfis)

IMPORTANTE:
- Nenhum comando é executado automaticamente: você deve rodar tudo manualmente.
- Nenhuma credencial real está incluída.
- Adapte PROJETO_ID, região, usuários, senhas e URLs ao seu ambiente real GCP.
