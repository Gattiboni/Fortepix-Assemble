FortePix – Build BETA da API (Simulação Segura)
===============================================

Este pacote representa o BUILD BETA da API FortePix em ambiente de simulação,
com foco em testes internos antes de qualquer exposição a usuários reais.

Inclui:
- Backend Java (Spring Boot) – núcleo principal em modo BETA
- Backend Python (FastAPI) – apoio, auditoria e testes
- Configurações para bancos em memória (H2 e SQLite)
- Estrutura preparada para futura integração com stack Docker e nuvem

IMPORTANTE:
- Nenhum dado real está incluído.
- Nenhuma conexão externa está configurada.
- A utilização deve ser feita apenas em ambiente controlado, interno e seguro.
