FortePix – Build Completo da API (Simulação Segura)
===================================================

Esta pasta contém o build completo de simulação da API FortePix:

- Backend Java (Spring Boot) – núcleo principal
- Backend Python (FastAPI) – apoio técnico, simulação e testes
- Configuração para bancos em memória (H2 e SQLite)

Nenhum dado real é incluído e nenhuma conexão externa é configurada.
